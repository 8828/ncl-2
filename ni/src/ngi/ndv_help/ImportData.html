Import Data File
  • From the DataVision File menu in the upper left hand corner, select Add File. The addfile dialog appears. The scrolling list labelled Files shows the data files in the default data directory.

  • Optionally choose another data directory by editing the Filter text field or by double clicking the left mouse button on an entry in the Directory list.

  • Select a data file by left clicking on a file name in the Files list. A suggested name that will be used to reference the file inside the DataVision environment appears in the bottom text entry field.

  • If desired, edit the suggested file name. In general short names are easier to use.

  • Press the OK button to add a single file. The dialog will disappear. If you want the window to stay around in order to add more than one file, use the Apply button instead.
