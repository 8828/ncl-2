Preview Data Files

  • From the DataVision File menu in the upper left hand corner, select Add File. The addfile dialog appears. The scrolling list labelled Files shows the data files in the default data directory.

  • Optionally choose another data directory by editing the Filter text field or by double clicking the left mouse button on an entry in the Directory list.

  • Hold down the right mouse button on a file name in the Files list entry to see global attributes associated with the data file. Double click the left mouse button on the file name to see the variables it contains in the Variables list

  • Once variables appear in the Variables list, hold down the left mouse button over a a variable name to see its dimensional information or the right mouse button to see its attributes.
