Plot Data

  • From the data variable page, press the Plot button to bring up a list of available plot styles.

  • Selecting a plot style causes the Create window to appear. You may modify the default plot name by editing the Name text entry field.

  • Press the Create pushbutton. A graphical display window will pop up in which the plot will will be drawn, unless...

  • the dimensionality of the data is insufficient, or more data is required (e.g. if only one component of a vector field is available.)
