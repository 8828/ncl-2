Select Data Variable

  • From the File Vars menu select the file and variable you want to work with. A page containing information about the variable appears.

  • To view the actual data values, press the Data toggle button. This brings up a scrollable two- dimensional spreadsheet-like view of the currently selected sub-region of the data (by default the fastest varying 2-d slice).

  • To select a different subregion of the data press the "Select a Subregion" toggle to bring up the subregion selector tool.
